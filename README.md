# SortAnalyzer

Aplikasi berbasis Streamlit untuk melakukan studi komparatif algoritma sorting dan binary search pada dataset mahasiswa.

## Fitur

* Bubble Sort
* Selection Sort
* Merge Sort
* Quick Sort
* Binary Search
* Benchmark Algoritma
* Analisis Kompleksitas Waktu
* Analisis Penggunaan Memori
* Dataset Mahasiswa Otomatis

## Teknologi

* Python
* Streamlit
* Pandas

## Menjalankan Program

Install dependensi:

pip install -r requirements.txt

Jalankan aplikasi:

streamlit run app.py
